# 医学教学病例智能体

这是一个采用文件优先架构、面向单人运行的医学教学病例 MVP，用于生成、审核和发布结构化病例及配套问题答案包。

## 当前状态

仓库安全基线和已批准的项目设计已经初始化。完成书面设计规格审阅后，将开始工程脚手架和详细实施计划。

## 仓库边界

申请材料、真实医学资料和运行产物仅保存在本地，不进入版本库。暂存或提交任何文件前，请先阅读仓库数据与隐私策略。

## 跨平台约定

项目以 Python 3.11+ 的模块命令作为 Windows 与 macOS 的统一运行入口。每台新设备克隆后都需要自行创建本机虚拟环境、`.local/sensitive_terms.txt`、`.env` 和私有数据目录；这些内容不会从 GitHub 获取。

仓库通过 GitHub Actions 在 `windows-latest` 与 `macos-latest` 上运行跨平台安全测试；S1 实施后，同一工作流将扩展为完整应用测试。
