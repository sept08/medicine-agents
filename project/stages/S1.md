# S1 最小运行骨架

## 目标

不使用 API Key 和真实医学资料，使用合成订单生成固定结构病例包并保存到本地文件仓库。

## 输入

- 完全合成的病例订单；
- 已冻结的领域 Schema；
- S0 项目与隐私基线。

## 输出

- Python 工程、严格领域模型和原子文件仓库；
- 固定合成生成工作流；
- CLI 和 FastAPI 接口；
- 跨平台安装、测试、运行入口及双平台 CI；
- 中文 PRD、质量评价方案、发布治理和医学输入指南。

## 可体验入口

```text
python scripts/test.py
python -m medicine_agents demo --order data/samples/orders/synthetic-order.json --data-dir data/runtime
python scripts/run_api.py
```

API 启动后访问 `http://127.0.0.1:8000/docs`。CLI 和 API 使用同一领域模型、生成工作流与文件仓库。
